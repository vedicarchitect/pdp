# add-supertrend-options-strategy

Intraday, paper-traded option-selling strategy driven by a universal **SuperTrend(3,1)**
indicator on the NIFTY index (5-minute bars), plus a **paper-trade journal** for daily
P&L and progress stats.

See `proposal.md` for the why/what, `design.md` for the approach, `tasks.md` for the
implementation checklist, and `specs/` for the capability deltas.
