## Why

The current frontend is a skeleton with basic functionality. To provide a professional, production-grade trading and investment platform, the UI needs significant aesthetic and usability improvements. This change aims to elevate the frontend design to a modern, dynamic, and premium standard, prioritizing rich aesthetics, seamless animations, and an intuitive user experience.

## What Changes

- Overhaul of the global design system (colors, typography, spacing) using modern principles (e.g., tailored dark mode, glassmorphism).
- Implementation of a sleek, modern UI with polished interactive elements.
- Refactoring existing React components (e.g., Shadcn UI elements) to use the new design system.
- Polishing critical user flows (Dashboard, OrderBook, Positions, Analytics).
- Enhancing responsive layouts and adding micro-animations for better engagement.

## Capabilities

### New Capabilities
- `frontend-design-system`: The core design system including tokens, utility classes, and base components for a premium look and feel.
- `ui-animations`: Subtle micro-animations for improved user interaction and feedback.

### Modified Capabilities
- `add-frontend-skeleton`: Upgrading the existing frontend skeleton to adhere to the new design standards.

## Impact

- `frontend/src/index.css` and related style files.
- `frontend/src/components/*` (refactoring for style and structure).
- `frontend/src/routes/*` (page layout updates).
- No backend API changes are expected.
